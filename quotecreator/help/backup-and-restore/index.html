Create a Backup of Your QuoteCreator License

9 min read

Learn how to create encrypted backups of your license and restore it on any device.

Why Create a Backup?

Your QuoteCreator license and data live in your browser. If you:

  • Clear your browsing history
  • Switch browsers or devices
  • Reinstall your operating system
  • Experience browser corruption

...you'll need to restore your license. Creating a backup protects your $12.99 investment.

Good News: You have TWO backup methods:

  1. Email - License key sent to your inbox (automatic)
  2. Encrypted File - .qclic backup file (recommended!)

Method 1: Email Backup (Automatic)

When you purchase, we automatically send your license key to your email.

Subject: "Your QuoteCreator License - QC-XXXX-XXXX-XXXX-XXXX"

What to do:

  • Keep this email - don't delete it!
  • Add to an "Important" or "Licenses" folder
  • Star/flag it for easy finding
  • Optional: Save to password manager

Recovery: Just copy/paste the key from email into Settings → License section

Pros:

  • Automatic - no setup required
  • Easy to find (search email for "QuoteCreator")
  • Works on any device

Cons:

  • Doesn't backup estimate counters
  • Relies on email not getting deleted

Method 2: Encrypted Backup File (Recommended)

Create a backup as an encrypted .qclic file.

What Gets Backed Up?

License Information:

  • License key
  • Activation date
  • Purchase details

Estimate Counters:

  • Daily estimate counts (so next estimate continues from correct number)
  • Last estimate number created
  • Last estimate date

Encryption:

  • Bank-level security (AES-256)
  • Password-protected
  • Only you can decrypt it

How to Create a Backup

Step 1: Go to Settings

  1. Open QuoteCreator
  2. Click the gear icon (Settings)
  3. Scroll down to Backup & Restore section

Step 2: Create a Backup

  1. Click "Create a Backup" button
  2. A window will appear asking for a password

Step 3: Choose a Strong Password

Requirements:

  • Minimum 8 characters
  • Use a password you'll remember!
❌ Weak: "password", "12345678" ✅ Strong: "MyQuote2025!", "PaintBiz#Secure"

Important: Write down this password or store in password manager. You'll need it to restore!

Step 4: Download the File

  1. Enter your password
  2. Click "Create Backup"
  3. File downloads: quotecreator-license-backup-20251101-123456.qclic

File name format: quotecreator-license-backup-YYYYMMDD-HHMMSS.qclic

  • Includes date and time when you created the backup

Step 5: Store Safely

Save your .qclic file in a secure location:

Recommended:

  • Cloud storage (Google Drive, Dropbox, iCloud)
  • Password manager (1Password, LastPass)
  • External hard drive
  • Email to yourself (as attachment)

Not Recommended:

  • Desktop only (gets deleted if computer fails)
  • USB drive only (can be lost)
  • Public/shared folders

Best Practice: Store in TWO places (cloud + password manager)

How to Restore from Backup

When to Restore?

Restore when:

  • You cleared browser data and lost your license
  • You're setting up QuoteCreator on a new device
  • You switched browsers
  • You reinstalled your OS

Step 1: Locate Your Backup File

Find your .qclic file:

  • Check Downloads folder
  • Check cloud storage (Google Drive, Dropbox, etc.)
  • Check email attachments
  • Check password manager

Step 2: Go to Settings

  1. Open QuoteCreator
  2. Click gear icon (Settings)
  3. Scroll to Backup & Restore section

Step 3: Restore from Backup

  1. Click "Restore from Backup" button
  2. A window appears

Step 4: Select File and Enter Password

  1. Click "Choose File" or drag-and-drop your .qclic file
  2. Enter the password you used when creating the backup
  3. Click "Restore"

Step 5: Verify Success

You should see:

  • "License restored successfully! Refreshing..."
  • "License activated! You now have unlimited estimates."

Check:

  • Go to Settings → License section
  • Should show "License Active"
  • Your license key should be displayed

Step 6: Create Next Estimate

Your estimate counter should continue from where you left off!

Example:
Last estimate before clearing data: EST-20251101-005
Next estimate after restore: EST-20251101-006 (if same day)

Set Up QuoteCreator on Multiple Devices

Goal: Use QuoteCreator on both your desktop and laptop with the same license

On Your Primary Device (Desktop):

  1. Activate your license (after purchase)
  2. Go to Settings → Backup & Restore
  3. Click "Create a Backup"
  4. Choose password: "MySecurePassword123"
  5. Save .qclic file to cloud storage (e.g., Google Drive)

On Your Secondary Device (Laptop):

  1. Open QuoteCreator (without license)
  2. Download .qclic file from cloud storage
  3. Go to Settings → Backup & Restore
  4. Click "Restore from Backup"
  5. Select the .qclic file
  6. Enter password: "MySecurePassword123"
  7. License activated on laptop!

Note: Estimates don't sync between devices automatically. Each device has its own estimates.

Troubleshooting Restore Issues

"Incorrect password or corrupted file"

Cause: Wrong password or damaged file

Fix:

  1. Double-check password (case-sensitive!)
  2. Try typing password instead of copy/paste
  3. Re-download the .qclic file from original location
  4. If still failing, use manual entry (use your license key from email)

"Invalid backup file format"

Cause: Wrong file type or corrupted

Fix:

  1. Verify file extension is .qclic
  2. Make sure you selected the correct file
  3. Don't rename the file
  4. If you edited the file, it's corrupted - use original

Restore succeeds but estimate counter doesn't restore

Cause: Backup was created before counter feature (Nov 1, 2025)

Explanation: Old backups don't include estimate counters. This is normal.

Fix: Your license still works! Estimates will use date-based numbering, so no duplicates.

Security & Privacy

Is my backup file secure?

Yes! Here's how:

Encryption: AES-256-GCM

  • Same encryption banks use
  • Requires password to decrypt
  • Even we can't decrypt it without your password

What if someone gets my file?

  • They can't open it without your password
  • Brute force would take millions of years
  • Store password separately from file

What's in the file?

  • License key
  • Estimate counters
  • Activation dates
  • NO estimates, NO client data, NO PDFs

Can QuoteCreator see my backup?

No!

  • Encryption happens in your browser
  • We never receive the file or password
  • Backup/restore is 100% local to your device

Common Questions

Q: How often should I create a backup?
A: Once right after purchase is sufficient. Create another backup if you've created many estimates and want counter backup.

Q: Can I have multiple backup files?
A: Yes! Each backup creates a new timestamped file. Keep the most recent one.

Q: What if I forget my backup password?
A: You can still recover using your license key from email. Just won't get counter restoration.

Q: Can I email my .qclic file to myself?
A: Yes! It's encrypted, so it's safe to email. Just remember the password.

Q: Will restoring overwrite my current estimates?
A: No. Restoring only restores your license and counters, not your estimates. Each device keeps its own estimates.

Q: Can I restore on multiple devices?
A: Yes! Restore the same .qclic file on as many devices as you want.

Q: What's better - email or file backup?
A: Both! Email is easier for license key recovery. File backup restores your counters too.
